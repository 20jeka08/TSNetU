Units
======================================

All data in WMOC is stored in the following SI (International System) units:

* Length = :math:`m`
* Diameter = :math:`m`
* Water pressure = :math:`m` (this assumes a fluid density of 1000 :math:`kg/m^3`)
* Elevation = :math:`m`
* Mass = :math:`kg`
* Time = :math:`s`
* Demand = :math:`m^3/s`
* Velocity = :math:`m/s`
* Acceleration = :math:`g` (1 :math:`g` = 9.8 :math:`m/s^2`)
* Volume = :math:`m^3`

When setting up analysis in WMOC, all input values should be specified in SI units. 
All simulation results are also stored in SI units.
